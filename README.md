# Task 3: Front-End Framework Basics (To-Do App)

## Objective
To build a functional To-Do List application that allows users to add, edit, complete, and delete tasks, showcasing an understanding of basic web framework features and database integration.

## Tools Used
* **Backend:** Python, Flask
* **Database:** MySQL
* **Frontend:** HTML5, CSS3 (Gradient UI design)
* **Development Environment:** VS Code

## Steps Performed
1. Set up a MySQL database (`todo_db`) with a `tasks` table to store task data, including completion status.
2. Configured a Flask backend (`app.py`) to handle routing and database connections.
3. Created CRUD (Create, Read, Update, Delete) endpoints to manage tasks.
4. Designed a modern, multi-page frontend (Home and Task Dashboard) using HTML/CSS.
5. Implemented dynamic Jinja templating to render tasks and toggle forms based on user interaction.

## Outcome
Successfully deployed a local, full-stack Task Management application where users can seamlessly add new tasks, edit existing ones, mark them as complete, and delete them, with all data persisting securely in a MySQL database.
